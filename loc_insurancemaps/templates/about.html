{% trans "About this Project" %}

The base of this platform is GeoNode, an open source geospatial content management system.

Two distinct custom components have been added:

  1. An interface and database configuration tailored specifically to the structure of this map series.
  2. A suite of tools that allows users to split, trim, and georeference the maps.

Included Volumes

To facilitate wide geographic and temporal coverages throughout Louisiana, while also limiting the overall number of volumes to make disk space manageable, the following criteria are applied to determine which volumes to include from the Library of Congress.

  • The earliest available edition for every community, regardless of date.
  • For New Orleans, only the earliest edition (1885).
  • For all other communities, all editions published through 1910.
