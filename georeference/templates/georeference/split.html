Check: Does this image need to be split before it can be georeferenced? Tell me more about this ↓

Sometimes, old maps will cover discontiguous areas, especially when the mapmakers were trying to fit a lot of content into a single page. In these cases, the image must be split into separate documents, and then each one can be georeferenced individually. Typically, you'll find strong black lines delineating different parts of the map. The image must be split along those lines.

This map must be split into four different pieces.

This map must be split into two pieces.

This map shows only one part of town, so it should not be split.

Feel free to inspect the map in the interface below, and then click "Don't Split" if the process is not necessary. You will then be brought to the georeferencing interface.

If the image does need to be split, draw lines across the interface and watch as the preview updates to show you how many new documents will be made.

  • Click once to start or continue a line—double-click to end it
  • You can the drawing at any time to start over
  • Click "Split!" when you are ready.
  • New documents will be made from each split part, and each one can be georeferenced separately.

Note: This evaluation is a very important initial step that should only be done once, so if you want to skip this step and jump right into georeferencing, start with documents that are ready for georeferencing.
